POO Básico

¿Qué es POO?

La Programación Orientada a Objetos (POO) es un paradigma de programación que organiza el código en "objetos". Estos objetos son instancias de clases, que actúan como plantillas para crear objetos.

Conceptos Clave

  • Clase: Una plantilla para crear objetos.
  • Objeto: Una instancia de una clase.
  • Atributos: Variables que almacenan datos del objeto.
  • Métodos: Funciones que definen el comportamiento del objeto.

Ejemplo en Código

class Persona {
    constructor(nombre, edad) {
        this.nombre = nombre;
        this.edad = edad;
    }

    saludar() {
        return `Hola, soy ${this.nombre} y tengo ${this.edad} años.`;
    }
}

const persona1 = new Persona("Juan", 25);
console.log(persona1.saludar()); // Muestra en la consola
return persona1.saludar(); // Muestra en la página

¿Cómo funciona el editor?

Escribe tu código en el editor y haz clic en Ejecutar para ver el resultado. Para mostrar un valor en la página, usa return. Por ejemplo: 

const saludo = "Hola, mundo!";
return saludo;

Si quieres depurar tu código o ver múltiples resultados, usa console.log. Los mensajes de console.log se mostrarán en la consola del navegador.

¿Cómo acceder a la consola del navegador?

Si usaste console.log, revisa la consola del navegador para ver los mensajes. Aquí te explicamos cómo acceder a ella:

  • Chrome: Presiona Ctrl + Shift + J (Windows/Linux) o Cmd + Option + J (Mac).
  • Firefox: Presiona Ctrl + Shift + K (Windows/Linux) o Cmd + Option + K (Mac).
  • Edge: Presiona Ctrl + Shift + I (Windows/Linux) o Cmd + Option + I (Mac).
  • Safari: Presiona Cmd + Option + C (Mac).
