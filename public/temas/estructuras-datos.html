Estructuras de Datos

¿Qué son las Estructuras de Datos?

Las estructuras de datos son formas de organizar y almacenar datos en una computadora para que puedan ser utilizados de manera eficiente. Algunas de las estructuras de datos más comunes son:

  • Arrays: Una colección de elementos almacenados en posiciones contiguas de memoria.
  • Listas Enlazadas: Una secuencia de nodos donde cada nodo contiene datos y una referencia al siguiente nodo.
  • Pilas: Una estructura LIFO (Last In, First Out) donde el último elemento añadido es el primero en ser eliminado.
  • Colas: Una estructura FIFO (First In, First Out) donde el primer elemento añadido es el primero en ser eliminado.
  • Árboles: Una estructura jerárquica donde cada nodo tiene un valor y referencias a nodos hijos.

Ejemplo de un Array en JavaScript

const frutas = ["Manzana", "Banana", "Naranja"];
console.log(frutas[0]); // Muestra "Manzana"
frutas.push("Uva"); // Añade "Uva" al final del array
console.log(frutas); // Muestra ["Manzana", "Banana", "Naranja", "Uva"]

¿Cómo funciona el editor?

Escribe tu código en el editor y haz clic en Ejecutar para ver el resultado. Para mostrar un valor en la página, usa return. Por ejemplo: 

const saludo = "Hola, mundo!";
return saludo;

Si quieres depurar tu código o ver múltiples resultados, usa console.log. Los mensajes de console.log se mostrarán en la consola del navegador.

¿Cómo acceder a la consola del navegador?

Si usaste console.log, revisa la consola del navegador para ver los mensajes. Aquí te explicamos cómo acceder a ella:

  • Chrome: Presiona Ctrl + Shift + J (Windows/Linux) o Cmd + Option + J (Mac).
  • Firefox: Presiona Ctrl + Shift + K (Windows/Linux) o Cmd + Option + K (Mac).
  • Edge: Presiona Ctrl + Shift + I (Windows/Linux) o Cmd + Option + I (Mac).
  • Safari: Presiona Cmd + Option + C (Mac).
