Programación Estructurada

¿Qué es la Programación Estructurada?

La programación estructurada es un enfoque que divide un problema grande en partes más pequeñas y manejables. Utiliza tres tipos principales de estructuras: secuencia, selección y repetición.

Conceptos Clave

  • Sentencias Secuenciales: Instrucciones que se ejecutan una tras otra.
  • Sentencias Condicionales: Permiten tomar decisiones (if, else).
  • Bucles: Repiten un bloque de código varias veces (for, while).
  • Funciones: Bloques de código reutilizables que realizan una tarea específica.

Ejemplo en Código


          // Función para obtener el saludo según la hora del día
          function obtenerSaludoPorHora() {
            const hora = new Date().getHours(); // Obtiene la hora actual del sistema
          
            if (hora < 12) {
              return "Buenos días";
            } else if (hora < 18) {
              return "Buenas tardes";
            } else {
              return "Buenas noches";
            }
          }
          
          // Función para saludar al usuario
          function saludar(nombre) {
            const saludo = obtenerSaludoPorHora(); // Llama a la función para obtener el saludo
            return `${saludo}, ${nombre}!`; // Devuelve el saludo con el nombre
          }
          
          // Llamada a la función para saludar a Iván
          const mensaje = saludar("Iván");
          
          // Mostrar el mensaje en la consola
          console.log(mensaje);

¿Cómo funciona el editor?

Escribe tu código en el editor y haz clic en Ejecutar para ver el resultado. Para mostrar un valor en la página, usa return. Por ejemplo: 

const saludo = "Hola, mundo!";
return saludo;

Si quieres depurar tu código o ver múltiples resultados, usa console.log. Los mensajes de console.log se mostrarán en la consola del navegador.

¿Cómo acceder a la consola del navegador?

Si usaste console.log, revisa la consola del navegador para ver los mensajes. Aquí te explicamos cómo acceder a ella:

  • Chrome: Presiona Ctrl + Shift + J (Windows/Linux) o Cmd + Option + J (Mac).
  • Firefox: Presiona Ctrl + Shift + K (Windows/Linux) o Cmd + Option + K (Mac).
  • Edge: Presiona Ctrl + Shift + I (Windows/Linux) o Cmd + Option + I (Mac).
  • Safari: Presiona Cmd + Option + C (Mac).
